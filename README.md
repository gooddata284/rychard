# Exemplo GitHub Pages com Imagens

Estrutura:
```
repo/
├─ index.html
└─ assets/
   ├─ css/style.css
   └─ img/
      ├─ logo.svg
      ├─ hero.svg
      ├─ gallery-1.svg
      ├─ gallery-2.svg
      └─ gallery-3.svg
```

Publicação:
1. Suba tudo para um repositório público.
2. Settings → Pages → Deploy from a branch → main → /(root).
3. Acesse `https://SEU_USUARIO.github.io/NOME_DO_REPO/`.
